# Sample Data

These files are small synthetic fixtures for documentation. They are not real scan results.

The target names use reserved example-style values, and the IP address uses `192.0.2.0/24`, which is reserved for documentation.

Files:

- `metadata.json`
- `assets.json`
- `live_hosts.json`
- `services.json`
- `technologies.json`
- `findings.json`

Use these files to understand expected parsed output shapes without scanning any real target.
